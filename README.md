# Dev NewTab
Расширение Firefox для создания своих закладок, без навязчивых новых "закладок" добавляемых firefox из истории.

![FirefoxExtentImage](https://github.com/zovdev/firefox-bookmarks/blob/main/FirefoxExtentImage.png?raw=true)

# Установка
- У вас должен быть `Firefox Developer Edition` или `Firefox Nightly`
- Перейти в поисковой строке на `about:config`
- Нажмите Принять риск и продолжить
- Написать в поиске `xpinstall.signatures.required` изменить на `false`
- Открываем `Расширения и темы (Ctrl+Shift+A)`
- Настройки, `Установить дополнение из файла`
- Выбрать `Все файлы (.)`, открыть `manifest.json`
